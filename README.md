## 🎯 **Sobre o Projeto**

O **Ideias para Criadores** é um aplicativo inovador desenvolvido por **Jairo Nunes**,  
voltado para ajudar **criadores de conteúdo** a superarem o bloqueio criativo.

Através da Inteligência Artificial, o app gera automaticamente:
- 🧠 **Títulos criativos**
- ⚡ **Hooks (ganchos de abertura)**
- 🎬 **Chamadas para ação (CTAs)**
- ✍️ **Descrições prontas para posts e vídeos**

---

## 🧩 **Como Funciona**

1️⃣ O usuário insere o **tema do conteúdo**  
2️⃣ Clica em **“Gerar Ideias”**  
3️⃣ Em segundos, recebe sugestões otimizadas, com base em IA.

## 🎬 **Vídeo Promocional**

Produzido em 5 cenas, com roteiro narrado pela voz IA **Brian (Clipchamp)**.  
Cada parte foi cuidadosamente sincronizada com as imagens criadas no DALL·E.

| Cena | Tema | Duração |
|------|------|---------|
| 1️⃣ | Abertura – O bloqueio criativo | 10s |
| 2️⃣ | Apresentação do aplicativo | 10s |
| 3️⃣ | Geração de ideias | 15s |
| 4️⃣ | Inspiração fluindo | 10s |
| 5️⃣ | Encerramento – “Crie. Inspire. Conecte.” | 15s |
